# mean2-course-seed
Seed project MEAN 2.0 - NodeJS, Angular 2, Express and MongoDB

Clone the repository:
- git clone https://github.com/atilla8huno/mean2-course-seed.git

Inside the project folder, run:
- npm install grunt -g
- npm install
- npm run grunt

From another terminal, run:
- node ./bin/www

Check localhost:
- http://localhost:3000/
